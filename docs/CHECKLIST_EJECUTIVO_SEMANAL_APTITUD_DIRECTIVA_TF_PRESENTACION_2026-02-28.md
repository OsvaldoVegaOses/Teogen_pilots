# Checklist Ejecutivo Semanal (Comite)

Uso: reunion de seguimiento de 20-30 minutos.  
Escala de control: `Verde` (cumple), `Amarillo` (riesgo), `Rojo` (desvio critico).

## Semana 1 - Alineacion de decision
1. Estan definidas las 3 decisiones directivas objetivo?
2. Estan definidos KPI y metas por decision?
3. Hay responsables formales por frente (Sponsor, Investigacion, Datos, Operaciones)?
4. Esta aprobada la matriz de evidencia requerida?

Regla:
1. `Verde` si 4/4.
2. `Amarillo` si 3/4.
3. `Rojo` si <=2/4.

## Semana 2 - Cobertura y contraste
1. Se completo muestra por actores clave?
2. Se incorporaron casos no adherentes/desconfirmatorios?
3. Se redujo sesgo de perspectiva promotora?
4. Se documento saturacion por categoria?

Regla:
1. `Verde` si 4/4.
2. `Amarillo` si 3/4.
3. `Rojo` si <=2/4.

## Semana 3 - Validacion causal y KPI
1. Cada proposicion prioritaria tiene evidencia robusta suficiente?
2. Cada proposicion tiene evidencia contradictoria analizada?
3. Consecuencias criticas tienen evidencia directa?
4. Existe tabla de impacto por KPI con nivel de confianza?

Regla:
1. `Verde` si 4/4.
2. `Amarillo` si 3/4.
3. `Rojo` si <=2/4.

## Semana 4 - Listo para decision
1. Deck ejecutivo final validado?
2. Recomendaciones priorizadas con impacto esperado?
3. Riesgos y mitigaciones aceptados por el comite?
4. Decision log completado (`Go`/`Pilot`/`No-Go`)?

Regla:
1. `Verde` si 4/4.
2. `Amarillo` si 3/4.
3. `Rojo` si <=2/4.

## Protocolo de escalamiento
1. Si una semana termina en `Rojo`, activar plan de recuperacion en 48h.
2. Si hay 2 semanas seguidas en `Amarillo`, re-priorizar alcance y recursos.
3. No avanzar a decision final con semaforo global menor a `Verde`.
